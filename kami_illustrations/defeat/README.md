# 敗北表情差分

勝敗画面で敗者側のカミに表示する、正方形の表情差分を置くフォルダです。

## 配置方法

通常表情の `kami_illustrations/` と同じファイル名で配置してください。

例:

```text
kami_illustrations/スサノオ.webp
kami_illustrations/defeat/スサノオ.webp
```

ファイル名は `kami_illustrations/manifest.json` の対応をそのまま利用するため、manifestを追加更新する必要はありません。

敗北表情が未配置、または読み込みに失敗した場合は、通常表情へ自動的にフォールバックします。

## 推奨仕様

- 形式: WebP
- 縦横比: 1:1
- 推奨サイズ: 512×512px
- 推奨容量: 1枚50～150KB程度
- 構図: 顔と上半身が中央に収まるもの

勝敗画面では敗者側だけ彩度と明度を少し落として表示します。

## 勝利セリフ

勝者のセリフは `index.html` の `KAMI_VICTORY_LINES` で管理します。

```js
"1": {
  "2": "スサノオがヤマトタケルに勝ったときのセリフ",
  "3": "スサノオがオオクニヌシに勝ったときのセリフ",
},
```

1段目が勝者のカミNo、2段目が敗者のカミNoです。同じカミ同士の組み合わせを除く全90通りを登録します。
